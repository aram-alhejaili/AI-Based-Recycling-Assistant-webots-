# AI-Based Recycling Assistant

## Overview
This project presents an intelligent robotic recycling system that uses Artificial Intelligence and Robotics technologies to classify and sort recyclable waste materials automatically.

The system was developed using the Webots simulation environment and integrates Deep Learning, Machine Learning, Computer Vision, and Reinforcement Learning techniques.

## Features
- Waste classification using CNN model
- SVM model comparison
- Reinforcement Learning using Q-Learning
- Webots robotic arm simulation
- Automated waste sorting process
- Real-time object detection and classification

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Scikit-learn
- Webots
- Reinforcement Learning (Q-Learning)

## Waste Categories
- Cardboard
- Metal
- Plastic
